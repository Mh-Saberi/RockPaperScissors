## How to Play
1. The player is prompted to choose between rock, paper, or scissors.
2. The computer makes a random choice.
3. The winner is determined based on the game rules:
   - Rock crushes scissors
   - Scissors cuts paper
   - Paper covers rock
4. The game continues until one player reaches 3 points.
